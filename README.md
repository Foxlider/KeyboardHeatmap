# Keyboard Heatmap

> Simple WPF keyboard heatmap in c#


---

## Installation

- Just execute KeyboardHeatmap.exe

### Clone

- Clone this repo to your local machine using `https://github.com/Foxlider/KeyboardHeatmap`

---

## Features

- Keayboard heatmap
- Handling QWERTY and AZERTY keymaps
- Data viewer
